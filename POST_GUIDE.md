# 📝 Adult Artist 網站發文全攻略 (給完全新手的白話文指南)

這是一份為你量身打造的超簡單發文教學！這套系統已經幫你把所有的排版美編都「自動化」了。只要照著這份指南做，你不用寫任何程式也能輕鬆發表一篇帶有超酷「暗黑科技風」與「駭客互動特效」的文章！

---

## 📍 第一步：創造新文章檔案

1. 打開你的專案資料夾。
2. 順著這個路徑一路點進去：`src/content/articles/`
3. 幫你的新文章建立一個新檔案。
   - ⚠️ **副檔名請一律使用 `.mdx`**。
   - 檔名建議一律使用「英文小寫」或是「數字」，例如 `my-first-post.mdx`、`2026-new-work.mdx`。

---

## ⚙️ 第二步：貼上「文章設定檔」 (Frontmatter)

每篇新文章的**最上方（必須是檔案的第一行）**，請一字不漏地貼上這區塊（被上下兩個 `---` 夾住），然後將裡面的內容改成你這篇文章的資訊：

```
---
title: "這是你的文章標題"
date: "2026-04-10"
category: "心得隨筆"
excerpt: "這是一段簡短的摘要，會出現在首頁。如果不填寫，系統會自動抓內文的頭100個字哦！"
isPinned: false
---
```

**💡 設定檔寫法小撇步：**
- **isPinned**: 如果填寫 `true`，這篇文章就會被強制釘在首頁的「# PINNED_ARTICLES」區。如果填寫 `false`，就會乖乖按照日期排在底下的「# RECENT_LOGS」近期文章區，並且自動分頁。
- **date**: 請務必按照 `YYYY-MM-DD` 的格式寫。系統會用這個日期來進行由新到舊的排序。

---

## ✍️ 第三步：開始寫內文 (常用語法教學)

過了設定檔之後，你就可以開始盡情寫文章了！我們使用的是 Markdown 語法，這是一種能讓你「邊打字、邊排版」的超強寫作方式。

不要擔心要怎麼寫 HTML，我們的系統已經設定好，會**自動**把你寫的以下語法，轉換成你最愛的字體大小與配色！

### 1. 標題 (Headings)
要幫小單元下大標題，只要在最前面加上 `#` 字號加一個「空白鍵」即可：
```
## # 這是一個大標題
### 這是一個中標題
```

### 2. 強調與標記文字 (Bold & Code)
這招在寫科技風文章超級好用！
```
如果你想要把某些文字 **加粗強調，變成亮青色**，請用兩個星星把它們包起來。

如果是想要營造駭客風，`像是這樣的專有名詞標記`，請用鍵盤左上角的 ` (Backtick) 把文字包起來，就會呈現帶橘色的駭客科技框！
```

### 3. 列點清單 (Lists)
請一定要記得：**符號後面一定要空一格！**
```
要列出重點可以這樣做：
- 這是平常無順序的第一項
- 這是平常無順序的第二項

如果是有邏輯順序的：
1. 睡覺
2. 吃飯
3. 翻譯
```

### 4. 引用區塊 (Blockquotes)
用大於符號 `>` 可以做出非常有書香味、旁邊帶有一條橘色質感粗線的唯美引用區塊：
```
> 這是一段引言。有時候遇到那種死活聽不懂的句子，我也只能雙手合十，開啟翻譯界最高境界的通靈模式。
```

## 🌟 第四步：召喚你的「專屬互動特效」(核心殺手鐧)

因為我們的副檔名是 `.mdx`，這代表你不只可以寫文字，你還能**跨越次元，直接在文章裡面召喚 React 程式特效**！

遇到適合的情境，毫不猶豫地把以下這些「短語法」丟到文章要顯示的行數就好：

### 📌 招式一：靈魂拷問組件
想要在文章結尾給讀者一個帶有動態展開、通往 Portaly 的震撼互動問卷？換個新行，直接打：
```
<SoulSearching />
```

### 📌 招式二：駭客亂碼解密文字
想要讓某一段精華句子，在使用者滑鼠往下滾動看到的瞬間，從亂碼漸漸解密成中文？
```
這是一句普通的文字，但遇到這句 <DecodeText text="會直接上演駭客亂碼解碼秀" />，是不是超帥！
```
*(注意文字要寫在 `text="..."` 的引號裡面呦)*

### 📌 招式三：終端機小圖示
如果想要在標題或是段落前面加個駭客專屬的 `>_` 終端機小圖示點綴，直接插入這句：
```
<Terminal /> 系統已啟動...
```
